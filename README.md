# 🔗 Linked List Python Utilities

Пакет високоефективних інструментів для роботи з однозв'язаними списками на Python. Модуль містить реалізації класичних алгоритмів, від базової маніпуляції вузлами до складних задач на розворот та детекцію циклів.

## Основні можливості

* **Створення та парсинг**: Генерація списків із рядків та візуалізація структур.
* **Алгоритми пошуку**: Отримання N-го елемента та вимірювання розміру циклу.
* **Трансформація**: Попарна перестановка (swap), розворот (reverse) та розділення списків.
* **Оптимізація**: Робота з відсортованими списками та видалення дублікатів.
